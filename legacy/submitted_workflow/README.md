# Initial-submission workflow archive

These six scripts reproduce the staged workbook workflow that accompanied the initial manuscript submission. They are retained unchanged for provenance and auditability.

They are **not** the recommended workflow for the revised manuscript. Peer review identified two assumptions that required correction:

1. both public VCFs use a combined Arabica reference, so revised analyses explicitly filter records by sgC/sgE pseudomolecule labels; and
2. annotation context is now restricted to direct sequence-alignment-supported blocks rather than length-scaled coordinate projection.

Use the scripts in the repository root `scripts/` directory for all revised analyses and publication assets.
