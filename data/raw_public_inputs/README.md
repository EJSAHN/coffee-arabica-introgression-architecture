# Public input files

Large public datasets are not distributed with this repository. Place or link the following files in a project directory before running the full workflow:

- `Accession_info.xlsx`
- `Arabica_sgC.TIP.BB.vcf.gz`
- `Arabica_sgE.TIP.BB.vcf.gz`
- `C_arabica_ET39_sgC.fasta`
- `C_arabica_ET39_sgE.fasta`
- `C_arabica_ET39_sgC.gff3`
- `C_arabica_ET39_sgE.gff3`

The direct-alignment workflow downloads the source assembly `GCA_036785775.1` and NCBI BLAST+ when they are not already cached under the project directory.
