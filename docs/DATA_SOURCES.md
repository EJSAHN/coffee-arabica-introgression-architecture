# Data sources

This workflow analyzes public *Coffea arabica* genomic resources.

## Dryad

Population-genomic data associated with Salojärvi et al. (2024):

- DOI: `10.5061/dryad.qnk98sfpt`
- URL: https://datadryad.org/dataset/doi:10.5061/dryad.qnk98sfpt

Files used from this resource include accession metadata, subgenome-resolved VCF files, syntenic alignments, and accompanying documentation.

## Zenodo

Filtered Coffea assemblies and annotations:

- DOI: `10.5281/zenodo.17250347`
- URL: https://zenodo.org/records/17250347

Files used from this resource include ET-39 subgenome GFF3 annotations and, optionally, corresponding FASTA assemblies.

## NCBI Datasets

Assembly metadata for *Coffea arabica* ET-39 v0.6:

- Assembly accession: `GCA_036785775.1`
- URL: https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_036785775.1/

The JSON Lines assembly report is used as assembly metadata context for annotation-coordinate reconciliation.
