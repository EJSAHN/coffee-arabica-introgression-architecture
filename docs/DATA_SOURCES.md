# Data sources

## Population-genomic resource

Salojärvi et al. Arabica population-genomic data on Dryad:

- DOI: `10.5061/dryad.qnk98sfpt`
- https://datadryad.org/dataset/doi:10.5061/dryad.qnk98sfpt

Files used include accession metadata and the sgC/sgE VCF resources. Both VCFs use a combined Arabica reference. Analyses therefore retain only pseudomolecules explicitly labelled for the intended subgenome and exclude opposite-subgenome and unassigned records.

## ET-39 assemblies and annotations

- DOI: `10.5281/zenodo.17250347`
- https://zenodo.org/records/17250347

Files used include the filtered ET-39 sgC/sgE FASTA assemblies and GFF3 annotations.
